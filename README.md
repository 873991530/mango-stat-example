如何运行？
--------

假设本地MySQL数据库的用户名和密码都为root

 * ``git clone https://github.com/jfaster/mango-stat-example.git``
 * ``cd mango-stat-example``
 * ``mvn clean jetty:run``
 * 浏览器访问：``localhost:8080/mango-stat``
